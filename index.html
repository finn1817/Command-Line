Terminal Help

Basic Commands

help
Display available commands
clear
Clear the terminal screen
ls [directory]
List directory contents
cd [directory]
Change directory
pwd
Print working directory

File Operations

cat [file]
Display file contents
vim/vi/nano [file]
Edit a file
touch [file]
Create a new file
mkdir [directory]
Create a new directory
rm [file]
Remove a file
download [file]
Download a file to your computer
upload
Upload a file from your computer

System & Information

whoami
Display current user
date
Display current date and time
weather
Show current weather
sysinfo
Display system information

Customization

theme [name]
Change terminal theme (dracula, monokai, solarized, github, nord)
animate [on/off]
Toggle terminal animations

Editor Commands

When in the editor:

ESC
Switch to command mode
i (in command mode)
Switch to insert mode
:w (in command mode)
Save file
:q (in command mode)
Quit editor
:wq (in command mode)
Save and quit

Tips

  • Use Up/Down arrow keys to navigate command history
  • Files with .txt extension can be downloaded to your computer
  • The terminal state is saved between sessions (using localStorage)
  • Try different themes with the 'theme' command
